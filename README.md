# 🖥 Rahul XP – Windows XP Style Portfolio

A fully interactive **Windows XP inspired developer portfolio** built with **React, TypeScript, and modern web technologies**.

This project recreates the nostalgic **Windows XP operating system experience inside the browser**, allowing users to explore projects, games, music, and applications through draggable windows and a desktop interface.

---

# ✨ Features

## 🟦 Windows XP Desktop Experience
- Authentic **Windows XP boot screen**
- Interactive **desktop environment**
- Draggable **window system**
- XP-style **taskbar and UI**

## 🎵 Built-in Music Player
- iPod-style music control wheel
- Album cover display
- Playlist support
- Next / previous / play / pause controls

## 🎮 Games Integration
- Classic browser games integrated inside XP-style windows  
Example: Minesweeper

## 📂 Portfolio Explorer
Projects and information displayed through **Explorer-style windows**.

## 🖌 XP Paint
Interactive **MS Paint style drawing application** inside a draggable window.

## 📄 Resume Viewer
Built-in **PDF viewer** to display the resume.

## 💻 Desktop Icons
Clickable icons to open applications like:
- Projects
- Music Player
- Games
- Paint
- Resume
- Contact

## 📺 CRT Screen Effect
Optional **retro CRT monitor overlay** effect to simulate an old computer display.
